2.어떤 경우 에러 발생하는지

```python
ZeroDivisionError = 숫자를 0으로 나누었을 때 나눌 수 없다는 에러
NameError    =  변수가 선언되어있지 않을 때
TypeError    = 작업을 수행하기 위한 타입이 정상적이지 못해 진행되지 않을 때
IndexError   = 해당 인덱스에 해당하는 공간이 설정되어있지 않을때
KeyError	 = dictionary 안에 찾고자하는 키의 값을 갖는 대상이 없을때
ModuleNotFoundError = 해당하는 이름을 갖고 있는 모듈이 없을 때
ImportError  = import 하고자하는 대상이 경로에 없을 때
```
